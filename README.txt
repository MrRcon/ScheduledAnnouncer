Introduction
------------

ScheduledAnnouncer is a easy to use plugin to auto announce broadcasts to 
users of your bukkit server.

For installation and usage help, see:
http://www.github.com/hohl/ScheduledAnnouncer

Requirements
------------

- Java 5+
- Recent version of Bukkit
